This is a merge of two CP 2077 mods by [djkovrik](https://www.nexusmods.com/cyberpunk2077/users/100354) with one additional feature.
1. [Declutter WorldMap](https://www.nexusmods.com/cyberpunk2077/mods/1930)
2. [Limited Fast Travel](https://www.nexusmods.com/cyberpunk2077/mods/1889)

Additional feature: All Fast Travel points are visible when fast traveling.

Summary: The World Map will show configured map pins along with the filtered fast travel points from Limited Fast Travel.  The Fast Travel Map will show all fast travel destinations, unfiltered.

Credits: djkovrik for two excellent mods, this just merges them into one mod.

Bugs: Mine.
